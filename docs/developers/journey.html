Guide for WoT Developers

There are many ways to be an expert in the Web of Things and below is our proposal on how to get there. This assumes no prior knowledge on any of the technologies the Web of Things builds upon. Since Web of Things bridges many fields such as Web development, embedded system development and Internet protocols, the preliminary sections below are tagged with the corresponding field and can be unfolded depending on your prior knowledge.

  1. JSON:
    1. Learning Sources
    2. Configuration files
    3. Exchanging data between devices (APIs)
    4. Metadata (e.g. API Descriptions)
  2. JSON Schema:
    1. Learning Sources
    2. Validation of JSON instances
    3. Generation of documentation, types for programming languages, UIs
  3. Node.js (optional)
    1. Explaining why (reference implementation)
    2. Node.js Tutorials
  4. JSON-LD:
    1. Learning Sources
    2. Bringing context to data, specifying the meaning (semantics) of the data
  5. A protocol, HTTP/REST is recommended:
    1. Learning Sources for the HTTP protocol
    2. How one programs a server (Thing)
    3. How one uses a REST Client and how one programs one
  6. Working with (constrained) Physical Things
    1. Understanding Device Categories
    2. Understanding lower level protocols (wireless and wired)
    3. Understanding how to work with sensors/li>
    4. Understanding how to work with actuators
    5. Example Thing development with Raspberry Pi
    6. Example Thing development with ESP32
  7. TD:
    1. Reading the spec
    2. Use a TD for a UI
    3. Interact with a Thing
    4. Existing Tooling
  8. Discovery:
    1. Introduction mechanisms
    2. Register TDs
    3. Find/Fetch TDs
    4. Existing implementations
  9. Scripting API:
    1. Idea behind how to program interactions with devices, the WoT way!
    2. Reading the specification
    3. node-wot
  10. Binding Templates:
    1. See how your knowledge in one protocol can apply to others
    2. See how other protocols work in WoT
  11. Profiles:
    1. Reduce implementation effort for your Consumers by implementing a profile
